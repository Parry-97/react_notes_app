# Notes / TODO app 
Simple Note taking app created using: 
* Mongo DB + Prisma
* React + React Query
* Node JS + Express JS
* Vite
* Fly.io

This is one of basic assignments from the highly recommended [!Full Stack Open 2022](https://fullstackopen.com/en) course created by the University of Helsinki
